# Low-Power-MCML-VCO-using-Stacking-Technique
1. Project OverviewThis project presents the design and characterization of a 3-stage MOS Current Mode Logic (MCML) based Ring Voltage-Controlled Oscillator (VCO) using GPDK 90nm technology. The design focuses on mitigating high static power consumption—a common drawback of MCML—by implementing a transistor stacking technique.
   
2. Key Design FeaturesArchitecture: 3-stage differential ring oscillator satisfying Barkhausen criteria for stable oscillation.Optimization: Implementation of a stacking technique where an additional NMOS transistor is connected in series with the tail current source to increase resistance and reduce current.Technology: GPDK 90nm CMOS process simulated in Cadence Virtuoso.Supply Voltage ($V_{dc}$): Operating range from 1.2V to 1.6V.

3. Performance SummaryParameterValueMax Frequency19.98 GHz (@ 1.6V $V_{dc}$) Tuning Range1.30 GHz to 19.98 GHz (via $V_{bias2}$) Power Consumption407.67 $\mu$W (@ 1.6V $V_{dc}$) Power Reduction23% improvement over conventional MCML design Best Phase Noise-63.03 dBc/Hz (@ 1.2V $V_{dc}$)

4. Design Analysis & MethodologyTransient Analysis: Confirmed periodic output waveforms and reliable startup behavior.Tuning Sensitivity:$V_{bias2}$ (Load Control): Primary tuning mechanism; increasing $V_{bias2}$ from 0.9V to 1.4V scales frequency down by increasing load resistance.$V_{dc}$ (Supply Control): Increasing $V_{dc}$ reduces delay stages, boosting frequency from 3.15 GHz to 19.98 GHz.$V_{bias1}$ (Tail Bias): Minimal frequency impact as increased current and voltage swing cancel each other, ensuring frequency stability.Noise Performance: Observed an inverse relationship where phase noise improves at lower operating speeds, demonstrating high stability at reduced frequencies.
